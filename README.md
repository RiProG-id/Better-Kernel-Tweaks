# Better-Kernel-Tweaks

![Better-Kernel-Tweaks Banner](module/banner.png)

The optimization module is designed to improve the efficiency of the Android kernel, ensuring increased stability.

## What is Better-Kernel-Tweaks?

Better-Kernel-Tweaks is a utility designed to enhance your device's performance by intelligently managing and optimizing kernel parameters. This ensures a smooth, lag-free experience, especially during resource-intensive tasks.

## Features

- **Automatic Optimization:** Applies a set of kernel and VM tweaks automatically at boot.
- **Wide Compatibility:** Supports a variety of rooted and modified Android environments.
- **Lightweight:** Runs once at boot and then exits, consuming no background resources.

## Compatibility

This tool is compatible with the following platforms:

- [Magisk](https://github.com/topjohnwu/magisk)
- [KernelSU](https://github.com/tiann/KernelSU)
- [APatch](https://github.com/bmax121/APatch)
- [AxManager (SU Mode)](https://github.com/fahrez182/AxManager)
- Other similar rooted environments.

## Installation

1.  Download the latest release from the [releases page](https://github.com/RiProG-id/Better-Kernel-Tweaks/releases).
2.  Flash the module through your preferred manager (e.g., Magisk, KernelSU).
3.  Reboot your device.
